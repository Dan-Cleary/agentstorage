# Contributing

Thanks for contributing to AgentStorage's public client surface.

## What belongs here

- CLI improvements under `packages/cli`
- API contract/docs updates under `openapi/` and docs files
- receiver helper scripts and examples

## What does not belong here

- Private backend/control-plane implementation details
- Production infrastructure/secrets/ops internals

## Pull request guidance

- Keep changes scoped and incremental.
- Include docs updates for any user-facing change.
- For API changes, update `openapi/agentstorage.v1.yaml` in the same PR.
